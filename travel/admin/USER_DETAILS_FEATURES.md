# Детальная информация о пользователе

## Новые возможности страницы деталей пользователя

### 📊 Статистика
- Подписчики
- Подписки
- Посты
- Альбомы
- Коммерческие посты
- Лайки отданные
- Лайки полученные
- Комментарии отданные
- Комментарии полученные
- Избранное

### 📑 Вкладки с детальной информацией

#### 1. Подписчики
Список пользователей, которые подписаны на данного пользователя:
- Аватар
- Имя и фамилия (ссылка на профиль)
- Email
- Дата подписки

#### 2. Подписки
Список пользователей, на которых подписан данный пользователь:
- Аватар
- Имя и фамилия (ссылка на профиль)
- Email
- Дата подписки

#### 3. Избранное
Посты и альбомы, которые пользователь добавил в избранное:
- Название
- Описание
- Дата добавления в избранное

#### 4. Лайкнутые посты ⭐ НОВОЕ
Все посты, которые пользователь лайкнул:
- Изображение поста
- Название и описание
- Автор поста
- Локация
- Дата лайка

#### 5. Комментарии ⭐ НОВОЕ
Все комментарии, которые оставил пользователь:
- Текст комментария
- Пост, к которому оставлен комментарий
- Автор поста
- Дата комментария

#### 6. Полученные ⭐ НОВОЕ
Активность других пользователей на постах данного пользователя:

##### 6.1. Лайки
Кто лайкнул посты пользователя:
- Аватар пользователя, который лайкнул
- Имя (ссылка на профиль)
- Какой пост был лайкнут (с превью)
- Дата лайка

##### 6.2. Избранное
Кто добавил посты пользователя в избранное:
- Аватар пользователя
- Имя (ссылка на профиль)
- Какой пост был добавлен (с превью)
- Дата добавления

##### 6.3. Комментарии
Комментарии к постам пользователя:
- Аватар комментатора
- Имя (ссылка на профиль)
- Текст комментария
- К какому посту (с превью)
- Дата комментария

## Использование

1. Откройте страницу пользователей: `/travel/admin/views/users.php`
2. Нажмите на кнопку "Просмотр" у любого пользователя
3. Изучите детальную информацию на вкладках

## Преимущества

- Нормализация URL изображений
- Лимит 50 записей на каждую категорию
- Сортировка по дате (новые первыми)
