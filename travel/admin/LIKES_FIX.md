# Исправление страницы лайков

## Проблемы и решения

### 1. ❌ Отсутствие данных на странице → ✅ ИСПРАВЛЕНО

**Проблемы:**
1. Неправильное название поля в SQL запросе: `loc.title` вместо `loc.name`
2. Неправильный порядок загрузки скриптов (likes.js загружался до jQuery)

**Решения:**
1. Исправлено поле в API: `loc.title` → `loc.name`
2. Исправлен порядок загрузки скриптов в likes.php:
   - jQuery загружается первым
   - Затем Bootstrap
   - Затем DataTables
   - Затем SweetAlert2
   - И наконец likes.js

## Тестирование

Должны увидеть:

- ✅ Сортировку
- ✅ Пагинацию
