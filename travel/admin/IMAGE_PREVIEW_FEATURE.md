# Функция просмотра изображений

## Обзор

Добавлена возможность просмотра изображений в полном размере при клике на превью во всех разделах страницы пользователя.

## Где работает

Функция просмотра изображений доступна в следующих разделах:

### 1. Посты
- Клик на превью поста открывает изображение в модальном окне
- Размер превью: 120x120px
- Полное изображение открывается в центре экрана

### 2. Альбомы
- Клик на обложку альбома открывает изображение
- Клик на любое фото внутри альбома также открывает его
- Размер обложки: 120x120px
- Размер фото в альбоме: 150px высота

### 3. Платные посты
- Клик на превью коммерческого поста
- Размер превью: 120x120px

### 4. Лайки
- **Поставленные лайки**: клик на превью лайкнутого поста
- **Полученные лайки**: клик на превью поста справа
- Размеры: 80x80px и 60x60px

### 5. Комментарии
- **Написанные комментарии**: клик на превью поста
- **Полученные комментарии**: клик на превью поста справа
- Размер: 60x60px

## Как это работает

### Визуальные индикаторы
- **Курсор**: меняется на pointer при наведении
- **Hover эффект**: изображение увеличивается на 5% с тенью
- **Tooltip**: "Нажмите для просмотра"

### Модальное окно
- Открывается по центру экрана
- Максимальная высота: 80vh
- Адаптивное под размер экрана
- Закрывается по клику на крестик или вне окна
- Поддерживает клавишу ESC

## Технические детали

### HTML структура
```html
<!-- Модальное окно -->
<div class="modal fade" id="imageModal">
    <div class="modal-dialog modal-dialog-centered modal-xl">
        <div class="modal-content">
            <div class="modal-header">
                <h5 class="modal-title">Просмотр изображения</h5>
                <button class="btn-close" data-bs-dismiss="modal"></button>
            </div>
            <div class="modal-body text-center">
                <img id="modalImage" src="" class="img-fluid">
            </div>
        </div>
    </div>
</div>
```

### CSS стили
```css
.preview-image {
    cursor: pointer;
    transition: transform 0.2s, box-shadow 0.2s;
}

.preview-image:hover {
    transform: scale(1.05);
    box-shadow: 0 4px 8px rgba(0,0,0,0.2);
}
```

### JavaScript функции

**showImageModal(imageSrc, title)**
- Открывает модальное окно с изображением
- Параметры:
  - `imageSrc` - URL изображения
  - `title` - заголовок модального окна (опционально)

**Обработчик кликов**
```javascript
$(document).on('click', '.preview-image', function(e) {
    e.preventDefault();
    const src = $(this).attr('src') || $(this).data('src');
    const title = $(this).attr('alt') || 'Просмотр изображения';
    showImageModal(src, title);
});
```

## Классы CSS

### .preview-image
Добавляется ко всем изображениям, которые можно просмотреть:
- Меняет курсор на pointer
- Добавляет hover эффект
- Регистрирует обработчик клика

## Примеры использования

### Пример 1: Просмотр поста
1. Откройте вкладку "Посты"
2. Наведите курсор на превью поста
3. Изображение увеличится и появится тень
4. Кликните на изображение
5. Откроется модальное окно с полным изображением

### Пример 2: Просмотр фото в альбоме
1. Откройте вкладку "Альбомы"
2. Нажмите кнопку "Фото" у альбома
3. Кликните на любое фото в сетке
4. Фото откроется в полном размере

### Пример 3: Просмотр платного поста
1. Откройте вкладку "Платные посты"
2. Кликните на превью любого поста
3. Изображение откроется в модальном окне

## Адаптивность

### Десктоп
- Модальное окно: размер XL (1140px)
- Изображение: максимум 80vh высоты
- Плавные анимации

### Мобильные устройства
- Модальное окно: на всю ширину экрана
- Изображение: адаптируется под ширину
- Сенсорное управление

## Производительность

- Изображения загружаются только при открытии модального окна
- Используется делегирование событий для оптимизации
- Плавные CSS transitions (0.2s)
- Нет дополнительных библиотек
