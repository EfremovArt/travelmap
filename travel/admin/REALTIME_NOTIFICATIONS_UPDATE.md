# Real-Time уведомления

## Обновления

Улучшена система уведомлений для более быстрого отображения новых публикаций и комментариев.

## Изменения

### 1. Увеличена частота обновлений

**Было:** 30 секунд  
**Стало:** 10 секунд

Теперь уведомления обновляются каждые 10 секунд вместо 30, что делает систему более отзывчивой.

### 2. Toast-уведомления

При появлении новых элементов показывается всплывающее уведомление (toast) в правом нижнем углу экрана:
- Показывает количество новых фото и комментариев
- Содержит кнопку для быстрого перехода к модерации
- Автоматически исчезает через 5 секунд
- Можно закрыть вручную

### 3. Анимация badge

При увеличении количества уведомлений badge "пульсирует" для привлечения внимания:
- Анимация длится 1 секунду
- Повторяется 2 раза
- Масштаб увеличивается до 1.3x

### 4. Отслеживание изменений

Система отслеживает предыдущее количество уведомлений и показывает toast только при появлении новых элементов, а не при каждом обновлении.

## Пользовательский опыт

1. **Без обновления страницы** - все обновления происходят автоматически
2. **Визуальная обратная связь** - анимация badge привлекает внимание
3. **Информативность** - toast показывает детали новых уведомлений
4. **Быстрый доступ** - кнопка в toast для перехода к модерации

## Производительность

- Интервал 10 секунд - баланс между отзывчивостью и нагрузкой на сервер
- Toast создаётся один раз и переиспользуется
- Анимации используют CSS transforms для лучшей производительности
