## Процесс удаления

### Удаление поста

```
1. Клик на [🗑️ Удалить]
   ↓
2. Подтверждение: "Вы уверены?"
   ↓
3. AJAX запрос к delete_post.php
   ↓
4. Транзакция в БД:
   - DELETE FROM likes WHERE photo_id = X
   - DELETE FROM comments WHERE photo_id = X
   - DELETE FROM favorites WHERE photo_id = X
   - DELETE FROM album_photos WHERE photo_id = X
   - DELETE FROM photos WHERE id = X
   ↓
5. Плавное исчезновение элемента
   ↓
6. Обновление статистики
   ↓
7. Сообщение: "Пост успешно удален"
```

### Удаление фото из альбома

```
1. Клик на [🖼️ Фото] → раскрытие списка
   ↓
2. Клик на [🗑️] под конкретным фото
   ↓
3. Подтверждение: "Вы уверены?"
   ↓
4. AJAX запрос к delete_album_photo.php
   ↓
5. DELETE FROM album_photos WHERE id = X
   ↓
6. Плавное исчезновение фото
   ↓
7. Обновление счетчика фото в альбоме
   ↓
8. Сообщение: "Фото удалено из альбома"
```

### Удаление альбома

```
1. Клик на [🗑️ Удалить]
   ↓
2. Подтверждение: "Вы уверены?"
   ↓
3. AJAX запрос к delete_album.php
   ↓
4. Транзакция в БД:
   - DELETE FROM album_likes WHERE album_id = X
   - DELETE FROM album_comments WHERE album_id = X
   - DELETE FROM album_favorites WHERE album_id = X
   - DELETE FROM album_photos WHERE album_id = X
   - DELETE FROM albums WHERE id = X
   ↓
5. Плавное исчезновение альбома
   ↓
6. Обновление статистики
   ↓
7. Сообщение: "Альбом успешно удален"
```

## API Endpoints

```
GET  /api/users/get_all_users.php
     ?search=текст&page=1&per_page=50
     → Список пользователей с фильтрацией

GET  /api/users/get_user_details.php
     ?user_id=123
     → Детальная информация о пользователе

GET  /api/posts/get_all_posts.php
     ?user_id=123&page=1&per_page=100
     → Посты пользователя

GET  /api/posts/get_all_albums.php
     ?user_id=123&page=1&per_page=100
     → Альбомы пользователя

GET  /api/posts/get_album_photos.php
     ?album_id=456
     → Фото в альбоме

POST /api/posts/delete_post.php
     post_id=789
     → Удаление поста

POST /api/posts/delete_album.php
     album_id=456
     → Удаление альбома

POST /api/posts/delete_album_photo.php
     album_photo_id=123
     → Удаление фото из альбома
```
