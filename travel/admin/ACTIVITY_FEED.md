# 📱 Лента активности - Unified Activity Feed

## Описание
Новая страница "Лента активности" объединяет лайки, комментарии и избранное в единую хронологическую ленту, похожую на Instagram. Показывает активность в режиме реального времени.

## Возможности

### 🎯 Единая лента
- Все типы активности в одном месте
- Хронологический порядок (новые сверху)
- Автообновление каждые 30 секунд

### 🔍 Фильтры
- **Все** - показывает всю активность
- **Лайки** - только лайки
- **Комментарии** - только комментарии  
- **Избранное** - только добавления в избранное

### 📊 Отображение активности

#### Лайк
```
[👤 Аватар + ❤️] Иван Иванов поставил(а) лайк на публикацию Петра Петрова
                  ⏰ 5 мин назад                                    [📷 Превью]
```

#### Комментарий
```
[👤 Аватар + 💬] Мария Сидорова прокомментировал(а) публикацию Ивана Иванова
                  "Отличное фото!"
                  ⏰ 10 мин назад                                   [📷 Превью]
```

#### Избранное
```
[👤 Аватар + ⭐] Петр Петров добавил(а) в избранное публикацию Марии Сидоровой
                  ⏰ 1 ч назад                                      [📷 Превью]
```

## Особенности

### ✨ Визуальные элементы
- **Аватар пользователя** с иконкой типа активности
- **Кликабельные имена** - ведут на профили пользователей
- **Превью публикации** - клик открывает детали поста
- **Относительное время** - "только что", "5 мин назад", "1 ч назад"

### 🔄 Автообновление
- Обновляется каждые 30 секунд (только на первой странице)
- Без перезагрузки страницы
- Плавная анимация при наведении

### 📄 Пагинация
- 50 записей на страницу
- Навигация по страницам
- Показывает текущую страницу

## API

### Endpoint
```
GET /travel/admin/api/activity/get_activity_feed.php
```

### Параметры
- `page` (optional) - номер страницы (default: 1)
- `per_page` (optional) - записей на страницу (default: 50, max: 100)
- `user_id` (optional) - фильтр по владельцу публикации
- `type` (optional) - тип активности: `all`, `like`, `comment`, `favorite`

## Преимущества перед отдельными страницами

### Было (3 отдельные страницы):
- ❌ Нужно переключаться между страницами
- ❌ Сложно отследить последовательность событий
- ❌ Нет единой картины активности

### Стало (единая лента):
- ✅ Вся активность в одном месте
- ✅ Хронологический порядок
- ✅ Быстрые фильтры
- ✅ Автообновление
- ✅ Похоже на Instagram/Facebook

## Использование

1. Откройте **Лента активности** в меню
2. Выберите фильтр (Все / Лайки / Комментарии / Избранное)
3. Просматривайте активность в хронологическом порядке
4. Кликайте на имена для перехода в профили
5. Кликайте на превью для просмотра публикации

## Технические детали

### SQL запросы
- 3 отдельных запроса (likes, comments, favorites)
- Объединение результатов в PHP
- Сортировка по `created_at DESC`
- Пагинация после сортировки

### Производительность
- Кэширование не требуется (данные меняются часто)
- Индексы на `created_at` для быстрой сортировки
- Лимит 50 записей на страницу

### Безопасность
- Требуется авторизация администратора
- Все изображения нормализуются
- XSS защита через `escapeHtml()`
