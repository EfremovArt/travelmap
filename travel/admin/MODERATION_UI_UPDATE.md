# Обновление интерфейса модерации

## Дата: 19 ноября 2025

## Внесенные изменения

### 1. Красные чекбоксы выбора
- Чекбоксы для выбора фотографий теперь имеют красный цвет (`accent-color: #dc3545`)
- Добавлен класс `.photo-checkbox-red` для стилизации
- Улучшена видимость с тенями и эффектами при наведении
- Размер увеличен до 22x22px для лучшей кликабельности

### 2. Кнопка удаления на каждой фотографии
- Добавлена кнопка корзины (🗑️) в правом верхнем углу каждой карточки фотографии
- Кнопка всегда видна на мобильных устройствах
- На десктопе появляется при наведении на карточку
- Позволяет быстро удалить фото без прокрутки страницы вверх

### 3. Функционал удаления
- Добавлена функция `deletePhoto(photoId)` в `moderation.js`
- При клике показывается подтверждающий диалог SweetAlert2
- После удаления:
  - Фото удаляется из базы данных
  - Удаляются связанные записи (лайки, комментарии, избранное)
  - Обновляется счетчик ожидающих модерации
  - Галерея перезагружается

### Измененные файлы

1. **travel/admin/assets/js/moderation.js**
   - Добавлен класс `photo-checkbox-red` к чекбоксам
   - Добавлена кнопка удаления в HTML разметку карточки
   - Добавлен обработчик клика для кнопки удаления
   - Добавлена функция `deletePhoto(photoId)`
   - Обновлен обработчик клика на карточку (игнорирует клик на кнопку удаления)

2. **travel/admin/assets/css/admin.css**
   - Добавлены стили для `.photo-checkbox-red`
   - Стили для checked состояния
   - Эффекты hover и focus
   - Адаптивность для мобильных устройств

### API эндпоинт

Используется существующий API:
- **URL**: `/travel/admin/api/moderation/delete_photo.php`
- **Метод**: POST
- **Параметры**: `{ photoId: number, csrf_token: string }`
- **Ответ**: `{ success: boolean, message: string }`

### Преимущества

1. **Улучшенная видимость**: Красные чекбоксы лучше заметны на фоне фотографий
2. **Быстрое удаление**: Не нужно прокручивать страницу вверх для удаления
3. **Удобство**: Все действия доступны прямо на карточке фотографии
4. **Мобильная оптимизация**: Кнопка удаления всегда видна на мобильных устройствах

### Совместимость

- ✅ Десктоп браузеры (Chrome, Firefox, Safari, Edge)
- ✅ Мобильные браузеры (iOS Safari, Chrome Mobile)
- ✅ Планшеты
- ✅ Адаптивный дизайн для всех разрешений экрана
