# 👥 Улучшение деталей поста - Лайки и комментарии

## Что добавлено

В модальном окне с деталями поста теперь отображаются:
- **Список пользователей, которые поставили лайк** (с аватарами и датой)
- **Список комментариев** (с аватарами, текстом и датой)

## Изменения

### 1. API: get_post_details.php

**Добавлено:**
```php
// Получаем список пользователей, которые поставили лайк
$likesStmt = $pdo->prepare("
    SELECT 
        u.id,
        CONCAT(u.first_name, ' ', u.last_name) as name,
        u.profile_image_url as image,
        l.created_at
    FROM likes l
    INNER JOIN users u ON l.user_id = u.id
    WHERE l.photo_id = :photo_id
    ORDER BY l.created_at DESC
    LIMIT 50
");

// Получаем список комментариев
$commentsStmt = $pdo->prepare("
    SELECT 
        c.id,
        c.comment_text as text,
        c.created_at,
        u.id as user_id,
        CONCAT(u.first_name, ' ', u.last_name) as user_name,
        u.profile_image_url as user_image
    FROM comments c
    INNER JOIN users u ON c.user_id = u.id
    WHERE c.photo_id = :photo_id
    ORDER BY c.created_at DESC
    LIMIT 50
");
```

**Возвращает:**
```json
{
  "success": true,
  "post": { ... },
  "likes": [
    {
      "id": 1,
      "name": "Иван Иванов",
      "image": "...",
      "created_at": "2025-01-01 12:00:00"
    }
  ],
  "comments": [
    {
      "id": 1,
      "text": "Отличное фото!",
      "user_id": 1,
      "user_name": "Иван Иванов",
      "user_image": "...",
      "created_at": "2025-01-01 12:00:00"
    }
  ]
}
```

### 2. Frontend: user_details.php

**Модальное окно теперь:**
- Увеличено до `modal-xl` (было `modal-lg`)
- Разделено на 2 колонки:
  - **Левая**: изображение и информация о посте
  - **Правая**: вкладки с лайками и комментариями
- Добавлены вкладки (tabs) для переключения между лайками и комментариями
- Каждый пользователь кликабелен - ведет на его профиль

**Отображение лайков:**
```
[Аватар] Имя пользователя
         Дата лайка
```

**Отображение комментариев:**
```
[Аватар] Имя пользователя  Дата
         Текст комментария
```

## Особенности

✅ **Лимит**: показывается до 50 лайков и 50 комментариев (последние)
✅ **Сортировка**: по дате (новые сверху)
✅ **Ссылки**: клик на имя пользователя открывает его профиль
✅ **Аватары**: с fallback на default-avatar.svg
✅ **Скролл**: если много лайков/комментариев, появляется прокрутка (max-height: 400px)
✅ **Адаптивность**: на мобильных колонки становятся вертикальными

## Где работает

### Посты:
- ✅ Вкладка "Посты" → кнопка "Просмотр"
- ✅ Вкладка "Избранное" → кнопка "Просмотр"
- ✅ Везде где есть кнопка просмотра поста

### Альбомы:
- ✅ Вкладка "Альбомы" → кнопка "Просмотр"
- ✅ Показывает лайки и комментарии по всем фотографиям альбома
- ✅ Под каждой фотографией показывается количество лайков и комментариев

## Измененные файлы

1. `travel/admin/api/posts/get_post_details.php` - добавлены запросы для лайков и комментариев поста
2. `travel/admin/api/posts/get_album_photos.php` - добавлены запросы для лайков и комментариев всех фотографий альбома
3. `travel/admin/views/user_details.php` - обновлены модальные окна с вкладками для постов и альбомов
