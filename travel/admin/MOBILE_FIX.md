# Исправление мобильной версии

## Проблемы

1. ❌ Текст выходит за границы экрана
2. ❌ Приходится скроллить влево-вправо
3. ❌ Длинные email не умещаются
4. ❌ Текст криво отображается

## Решения

### 1. Предотвращение горизонтального скролла

```css
body {
    overflow-x: hidden;
}

.container-fluid {
    overflow-x: hidden;
}
```

### 2. Перенос длинных слов

```css
.card-body, .table td, small, p, h6 {
    word-wrap: break-word;
    word-break: break-word;
    overflow-wrap: break-word;
    hyphens: auto;
}
```

Теперь длинные email и текст будут переноситься на новую строку.

### 3. Flex элементы с min-width

```css
.flex-grow-1 {
    min-width: 0;
    overflow: hidden;
}
```

Это предотвращает выход flex элементов за границы.

### 4. Компактные размеры

**Карточки статистики:**
- Desktop: 4 колонки
- Mobile: 2 колонки (col-6)
- Уменьшенные отступы (py-2)

**Изображения:**
- Desktop: 120x120px
- Mobile: 80x80px
- Very Small: 60x60px

**Шрифты:**
- Заголовки: 1.25rem → 1.1rem
- Текст: 0.875rem → 0.75rem
- Мелкий текст: 0.75rem → 0.65rem

### 5. Адаптивные кнопки

```css
.border.rounded .d-flex button {
    font-size: 0.7rem;
    padding: 0.25rem 0.4rem;
    white-space: nowrap;
}
```

Кнопки компактнее, но текст не переносится.
