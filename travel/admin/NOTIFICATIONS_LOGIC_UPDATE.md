# Обновление логики уведомлений

## Изменения

Изменена логика подсчета новых фото и комментариев для уведомлений.

## Старая логика

**Фото:** Показывались только фото со статусом `pending` или `NULL`  
**Комментарии:** Показывались только комментарии к фото

**Проблема:** Если фото публикуются сразу без модерации (без pending статуса), уведомления не показывались.

## Новая логика

### Фото
- Показываются все фото, созданные после последнего просмотра
- Если админ никогда не просматривал - показываются фото за последние 24 часа
- Статус модерации не учитывается

### Комментарии
- Показываются комментарии к фото И к альбомам
- Считаются комментарии, созданные после последнего просмотра
- Если админ никогда не просматривал - показываются комментарии за последние 24 часа

## Код

### API: get_new_counts.php

```php
// Новые фото
if ($lastPhotoView) {
    // После последнего просмотра
    SELECT COUNT(*) FROM photos WHERE created_at > :last_view
} else {
    // За последние 24 часа
    SELECT COUNT(*) FROM photos WHERE created_at > DATE_SUB(NOW(), INTERVAL 24 HOUR)
}

// Новые комментарии (фото + альбомы)
if ($lastCommentView) {
    // Комментарии к фото
    SELECT COUNT(*) FROM comments WHERE created_at > :last_view
    // Комментарии к альбомам
    SELECT COUNT(*) FROM album_comments WHERE created_at > :last_view
} else {
    // За последние 24 часа
    SELECT COUNT(*) FROM comments WHERE created_at > DATE_SUB(NOW(), INTERVAL 24 HOUR)
    SELECT COUNT(*) FROM album_comments WHERE created_at > DATE_SUB(NOW(), INTERVAL 24 HOUR)
}
```

## Преимущества

1. **Работает без pending статуса** - фото публикуются сразу, уведомления всё равно показываются
2. **Учитываются комментарии к альбомам** - полная картина активности
3. **Разумное начальное значение** - 24 часа вместо всех записей
4. **Отслеживание просмотров** - после просмотра счетчик сбрасывается

## Как работает

1. Пользователь загружает фото → фото создаётся с `created_at = NOW()`
2. Админ открывает любую страницу → через 10 секунд API проверяет новые фото
3. Если есть фото новее `last_viewed_at` → показывается уведомление
4. Админ открывает модерацию → вызывается `mark_as_viewed.php`
5. Обновляется `last_viewed_at = NOW()` → счетчик сбрасывается

2. Нажмите "Очистить просмотры"
3. Проверьте, что показываются фото за последние 24 часа
