# Система счетчиков модерации

## Описание
Система отслеживает просмотренные элементы и показывает только количество новых (непросмотренных) фото и комментариев.

## Как работает

### 1. Отслеживание просмотров
- При открытии вкладки "Фотографии" или "Комментарии" система автоматически отмечает время просмотра
- Отметка происходит через 1 секунду после открытия вкладки (чтобы пользователь успел увидеть контент)

### 2. Подсчет новых элементов
- **Фотографии**: считаются фото со статусом pending, созданные после последнего просмотра
- **Комментарии**: считаются комментарии, созданные после последнего просмотра

### 3. Визуальные индикаторы
- Если есть новые элементы - бейдж красный (bg-danger)
- Если новых нет - бейдж серый (bg-secondary)
- Счетчик показывает точное количество новых элементов

## Установка

1. Примените миграцию:
```
https://bearded-fox.ru/travel/admin/apply_views_tracking.php
```

2. Таблица `admin_views` будет создана автоматически при первом использовании

## API

### GET /api/moderation/get_new_counts.php
Возвращает количество новых фото и комментариев для текущего админа

**Ответ:**
```json
{
  "success": true,
  "counts": {
    "newPhotos": 5,
    "newComments": 12,
    "lastPhotoView": "2024-01-15 10:30:00",
    "lastCommentView": "2024-01-15 09:15:00"
  }
}
```

### POST /api/moderation/mark_as_viewed.php
Отмечает просмотр фото или комментариев

**Параметры:**
- `view_type`: "photos" или "comments"

**Ответ:**
```json
{
  "success": true,
  "message": "Просмотр отмечен"
}
```

## Обновление счетчиков
- Автоматически при загрузке страницы
- Каждые 2 минуты
- После отметки просмотра
