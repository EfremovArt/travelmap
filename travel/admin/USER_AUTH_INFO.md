# Информация о методе авторизации пользователя

## Что добавлено

В профиле пользователя теперь отображается:
1. **Метод авторизации** - как пользователь зарегистрировался
2. **Дата рождения** - с автоматическим расчетом возраста

## Методы авторизации

### 1. Apple ID 🍎
- Отображается если есть `apple_id`
- Показывается иконка Apple
- Дополнительная строка с Apple ID

### 2. Email 📧
- Отображается если есть email с символом `@`
- Показывается иконка конверта
- Дополнительная строка с email

### 3. Телефон 📞
- Отображается если есть `phone_number`
- Показывается иконка телефона
- Дополнительная строка с номером телефона

### 4. Google
- Отображается если есть email без Apple ID
- Показывается иконка Google
- Дополнительная строка с email

### 5. Неизвестно ❓
- Если нет ни одного из вышеперечисленных

## Дата рождения

Если у пользователя указана дата рождения:
- Отображается в формате ДД.ММ.ГГГГ
- Автоматически рассчитывается возраст
- Показывается в формате: `01.01.1990 (34 года)`

## Структура данных

### API Response
```json
{
  "user": {
    "id": 1,
    "firstName": "Иван",
    "lastName": "Иванов",
    "email": "ivan@example.com",
    "appleId": null,
    "phoneNumber": null,
    "dateOfBirth": "1990-01-01",
    "profileImage": "...",
    "createdAt": "2024-01-01 12:00:00"
  }
}
```

### Поля в таблице users
- `email` - email пользователя
- `apple_id` - Apple ID (если авторизация через Apple)
- `phone_number` - номер телефона
- `date_of_birth` - дата рождения (DATE)

## Логика определения метода авторизации

```javascript
if (user.appleId) {
    // Apple ID
} else if (user.email && user.email.includes('@')) {
    // Email
} else if (user.phoneNumber) {
    // Телефон
} else if (user.email) {
    // Google
} else {
    // Неизвестно
}
```

## Технические детали

### Расчет возраста
```javascript
const birthDate = new Date(user.dateOfBirth);
const today = new Date();
let age = today.getFullYear() - birthDate.getFullYear();
const monthDiff = today.getMonth() - birthDate.getMonth();
if (monthDiff < 0 || (monthDiff === 0 && today.getDate() < birthDate.getDate())) {
    age--;
}
```
