# Краткая инструкция по новой админке

## Что изменилось

### ✅ Поиск пользователей
Теперь можно искать по:
- Имени
- Фамилии  
- Email
- Apple ID

Просто введите любое из этих значений в поле поиска на странице "Пользователи".

### ✅ Страница пользователя

При клике на пользователя открывается детальная страница с вкладками:

#### 1. **Посты**
- Список всех постов пользователя
- Кнопка "Удалить" для каждого поста
- Показывает превью, название, описание, локацию, количество лайков и комментариев

#### 2. **Альбомы**
- Список всех альбомов
- Кнопка "Фото" - раскрывает список фото в альбоме
- Можно удалить:
  - Отдельное фото из альбома (кнопка под каждым фото)
  - Весь альбом целиком (кнопка "Удалить" рядом с альбомом)

#### 3. **Платные посты**
- Список всех коммерческих постов пользователя
- Показывает статус (активен/неактивен) и тип (фото/альбом/отдельный)
- Можно:
  - Просмотреть детали (кнопка "Детали")
  - Активировать/деактивировать пост
  - Удалить платный пост

#### 4. **Подписчики**
- Кто подписан на этого пользователя
- Клик по имени → переход на профиль подписчика

#### 5. **Подписки**
- На кого подписан пользователь
- Клик по имени → переход на профиль

#### 6. **Лайки**
Две подвкладки:
- **Поставленные** - какие посты лайкнул пользователь
- **Полученные** - кто лайкнул посты пользователя

#### 7. **Комментарии**
Две подвкладки:
- **Написанные** - комментарии пользователя
- **Полученные** - комментарии к постам пользователя

## Как пользоваться

### Найти пользователя
1. Откройте раздел "Пользователи" в боковом меню
2. Введите имя, email или Apple ID в поле поиска
3. Нажмите на имя пользователя в таблице

### Удалить пост
1. Откройте профиль пользователя
2. Перейдите на вкладку "Посты"
3. Нажмите красную кнопку "Удалить" рядом с постом
4. Подтвердите удаление

### Удалить фото из альбома
1. Откройте профиль пользователя
2. Перейдите на вкладку "Альбомы"
3. Нажмите кнопку "Фото" у нужного альбома
4. Нажмите "Удалить" под конкретным фото
5. Подтвердите удаление

### Удалить весь альбом
1. Откройте профиль пользователя
2. Перейдите на вкладку "Альбомы"
3. Нажмите красную кнопку "Удалить" рядом с альбомом
4. Подтвердите удаление

### Управление платными постами
1. Откройте профиль пользователя
2. Перейдите на вкладку "Платные посты"
3. Для просмотра деталей нажмите "Детали"
4. Для изменения статуса нажмите "Активировать" или "Деактивировать"
5. Для удаления нажмите "Удалить" и подтвердите

### Посмотреть подписчиков/подписки
1. Откройте профиль пользователя
2. Перейдите на вкладку "Подписчики" или "Подписки"
3. Нажмите на имя любого пользователя для перехода к его профилю

## Важно

- Все удаления необратимы
- При удалении поста удаляются все его лайки и комментарии
- При удалении альбома удаляются все связи, но сами фото остаются
- При удалении фото из альбома, само фото остается в базе данных
- При удалении платного поста удаляются все связи
- Деактивация платного поста не удаляет его, а только скрывает
- Статистика обновляется автоматически после удаления

## Безопасность

- Все действия требуют авторизации администратора
- Перед удалением всегда запрашивается подтверждение
- Все операции логируются в базе данных
