# Руководство по использованию мобильной версии админ-панели

## Навигация на мобильных устройствах

### Открытие меню
1. Нажмите на кнопку с тремя полосками (☰) в левом верхнем углу
2. Откроется боковое меню со всеми разделами
3. Выберите нужный раздел
4. Меню автоматически закроется после выбора

### Работа с таблицами

#### Выбор количества строк
- Выпадающий список "Показать 25, 50, 100 строк" теперь корректно отображается
- Стрелочка не перекрывает число
- На мобильных устройствах список компактнее

#### Прокрутка таблиц
- На мобильных устройствах таблицы можно прокручивать горизонтально
- Проведите пальцем влево/вправо для просмотра всех колонок

#### Поиск и фильтры
- Поле поиска адаптировано под мобильные экраны
- Фильтры располагаются вертикально для удобства

### Советы по использованию

1. **Портретная ориентация** - лучше всего подходит для просмотра списков
2. **Ландшафтная ориентация** - удобна для работы с таблицами (больше колонок видно)
3. **Кнопки действий** - на мобильных расположены вертикально для удобства нажатия
4. **Изображения** - автоматически уменьшаются на маленьких экранах
