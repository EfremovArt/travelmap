# Исправление отображения комментариев в модерации

## Проблема
В разделе модерации → комментарии не везде отображался пост/альбом, к которому был написан комментарий. В профиле пользователя всё отображалось правильно.

## Причина
API модерации (`get_all_comments.php`) получал только комментарии к фотографиям из таблицы `comments`, но не получал комментарии к альбомам из таблицы `album_comments`. 

В то время как API комментариев пользователя (`api/comments/get_all_comments.php`) использовал UNION для получения обоих типов комментариев.

## Решение

### 1. API модерации (get_all_comments.php)
Обновили запрос для получения комментариев обоих типов:

**Было:**
```sql
SELECT ... FROM comments c
LEFT JOIN users u ON c.user_id = u.id
LEFT JOIN photos p ON c.photo_id = p.id
```

**Стало:**
```sql
SELECT * FROM (
    (SELECT ... FROM comments c ... -- комментарии к фото
     LEFT JOIN photos p ON c.photo_id = p.id)
    UNION ALL
    (SELECT ... FROM album_comments ac ... -- комментарии к альбомам
     LEFT JOIN albums a ON ac.album_id = a.id)
) as all_comments
```

### 2. JavaScript (moderation.js)

#### Обновлено отображение комментариев:
- Добавлена поддержка отображения альбомов
- Для фото: показывается превью и название
- Для альбомов: показывается иконка коллекции и название
- Добавлен атрибут `data-comment-type` для определения типа

#### Добавлена функция просмотра альбома:
```javascript
function viewAlbumDetailsFromComment(albumId) {
    window.location.href = `posts.php?album_id=${albumId}`;
}
```

#### Обновлена функция удаления:
```javascript
function deleteComment(commentId, commentType = 'photo') {
    // Теперь принимает тип комментария
}
```

## Изменения в коде

### Файлы:
1. `travel/admin/api/moderation/get_all_comments.php` - UNION запрос для обоих типов
2. `travel/admin/assets/js/moderation.js` - отображение и обработка обоих типов

### Новые поля в ответе API:
- `comment_type` - тип комментария ('photo' или 'album')
- `album_id` - ID альбома (если комментарий к альбому)
- `photo_id` - ID фото (если комментарий к фото)

## Результат

✅ Теперь в модерации отображаются все комментарии:
- Комментарии к фотографиям (с превью)
- Комментарии к альбомам (с иконкой)

✅ Можно кликнуть на пост/альбом для просмотра деталей

✅ Удаление работает для обоих типов комментариев

## Тестирование

1. Откройте раздел Модерация → Комментарии
2. Проверьте, что отображаются комментарии к фото (с превью)
3. Проверьте, что отображаются комментарии к альбомам (с иконкой)
4. Кликните на пост - должен открыться модал с деталями
5. Кликните на альбом - должна открыться страница постов с фильтром по альбому
6. Попробуйте удалить комментарий обоих типов
