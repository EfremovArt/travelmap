# Улучшения админ-панели

## Обзор изменений

Админ-панель была переработана для более удобного управления пользователями и их контентом.

## Основные улучшения

### 1. Расширенный поиск пользователей

**Что изменилось:**
- Поиск теперь работает по имени, фамилии, email И Apple ID
- Добавлено отображение Apple ID в таблице пользователей
- Улучшена визуализация: если у пользователя нет email, показывается Apple ID с иконкой

**Файлы:**
- `api/users/get_all_users.php` - обновлен SQL запрос для поиска
- `views/users.php` - обновлен заголовок колонки
- `assets/js/users.js` - добавлена логика отображения Apple ID

### 2. Улучшенная страница детальной информации пользователя

**Новая структура вкладок:**

1. **Посты** - все посты пользователя с возможностью удаления
2. **Альбомы** - все альбомы с возможностью:
   - Просмотра фото в альбоме
   - Удаления отдельных фото из альбома
   - Удаления всего альбома
3. **Подписчики** - список подписчиков с переходом на их профили
4. **Подписки** - список тех, на кого подписан пользователь
5. **Лайки** - разделено на:
   - Поставленные (посты, которые лайкнул пользователь)
   - Полученные (кто лайкнул посты пользователя)
6. **Комментарии** - разделено на:
   - Написанные (комментарии пользователя)
   - Полученные (комментарии к постам пользователя)

**Файлы:**
- `views/user_details.php` - полностью переработана структура
- `api/users/get_user_details.php` - добавлено поле apple_id

### 3. Управление контентом

**Новые возможности:**

#### Удаление постов
- Кнопка удаления рядом с каждым постом
- Подтверждение перед удалением
- Автоматическое обновление статистики после удаления

#### Управление альбомами
- Просмотр всех фото в альбоме (раскрывающийся список)
- Удаление отдельных фото из альбома
- Удаление всего альбома целиком
- Визуальная сетка фото с превью

**Новые API endpoints:**
- `api/posts/delete_post.php` - удаление поста
- `api/posts/delete_album.php` - удаление альбома
- `api/posts/delete_album_photo.php` - удаление фото из альбома

## Технические детали

### Безопасность
- Все операции удаления требуют аутентификации администратора
- Используются транзакции для удаления связанных данных
- Подтверждение действий через JavaScript

### Удаление данных

При удалении поста удаляются:
- Лайки к посту
- Комментарии к посту
- Записи в избранном
- Связи с альбомами
- Сам пост

При удалении альбома удаляются:
- Лайки альбома
- Комментарии к альбому
- Записи в избранном
- Связи с фото (сами фото остаются)
- Сам альбом

При удалении фото из альбома:
- Удаляется только связь (запись в album_photos)
- Само фото остается в базе данных

## Использование

### Поиск пользователей
1. Перейдите в раздел "Пользователи"
2. Введите имя, фамилию, email или Apple ID в поле поиска
3. Результаты обновятся автоматически

### Просмотр деталей пользователя
1. Нажмите на имя пользователя в таблице
2. Откроется страница с полной информацией
3. Используйте вкладки для навигации по разным разделам

### Управление постами
1. Перейдите на вкладку "Посты"
2. Нажмите кнопку "Удалить" рядом с нужным постом
3. Подтвердите действие

### Управление альбомами
1. Перейдите на вкладку "Альбомы"
2. Нажмите "Фото" чтобы раскрыть список фото в альбоме
3. Удалите отдельные фото или весь альбом целиком

### Просмотр подписчиков/подписок
1. Перейдите на соответствующую вкладку
2. Нажмите на имя пользователя для перехода к его профилю
