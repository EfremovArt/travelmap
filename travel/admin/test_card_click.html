На проверке

Проверка:

  • ✅ Клик на изображение → открывает модальное окно
  • ✅ Клик на заголовок → открывает модальное окно
  • ✅ Клик на текст → открывает модальное окно
  • ✅ Клик на badge → открывает модальное окно
  • ❌ Клик на чекбокс → НЕ открывает модальное окно (только выбирает)
  • ❌ Клик на кнопку одобрить → НЕ открывает модальное окно (только одобряет)
  • ❌ Клик на кнопку отклонить → НЕ открывает модальное окно (только отклоняет)
